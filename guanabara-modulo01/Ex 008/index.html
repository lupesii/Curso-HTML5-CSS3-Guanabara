Principais formatações

Negrito / Destaque

Não utilize o Negrito de maneira não-semântica, usando as tags <B>

Utilize o Negrito de maneira semântica, usando as tags <STRONG>

Itálico / Ênfase

Não utilize o ítalico de maneira não-semântica, usando as tags <I>

Utilize o ítalico de maneira semântica, usando as tags <EM>

Texto marcado

Podemos marcar o texto utilizando a tag <MARK>.

Para mudar a cor do marcador basta utiliza o CSS com <style>.

Texto grande e pequeno

Podemos criar um texto grande e um texto pequeno

Texto deletado

É possivel marcar um texto que foi excluido, sendo assim pode ser lido, mas não deve ser considerado. Para isso utiliza-se as tags <del>

Texto inserido

É possivel marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Para isso utiliza-se as tags <ins>

Também existe a tag <u>, porém ela é não-semântica

Texto sobrescrito

Para inserir coisas do tipo x20+3.

Texto subescrito

Para inserir coisas do tipo H20
